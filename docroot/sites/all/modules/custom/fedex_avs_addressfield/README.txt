DESCRIPTION

Provides FedEx AVS integration with the address field module.

HOW IT WORKS

- When a user submits an address, it is validated with FedEx AVS.
- If the address is invalid, a detailed error message is shown,
  along with an option to bypass the validation and continue.

INSTALLATION

- Install the fedex_avs module and configure it correctly.
- Install this module
- Configure the fields you wish to validate and enable the "Fedex Address Validation"
  setting.
